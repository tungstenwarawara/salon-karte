# UXレビュー チェックリスト

## 対象ユーザー
ITリテラシーが高くない個人サロンオーナー（1人運営）。スマホメイン。

## 業務フロー整合
- この機能はサロン業務のどの場面で使われるか？（来店中 / 来店前後 / 管理業務）
- その場面で自然にたどり着ける導線があるか？
- 「この画面の前に何をしていた？後に何をする？」を確認

## 認知負荷
- 1タスク完了に必要な判断（選択・入力）は3つ以下か？
- ユーザーが「どこを押せばいいかわからない」状態にならないか？
- 説明テキストなしでボタン名・ラベルの意味が伝わるか？

## 入力手段の最適化
- 手入力が必要な箇所で、マスタデータからのプリフィル/ドロップダウンが使えないか？
- 繰り返し入力される値（メニュー名、金額等）がテンプレート化されているか？
- 物販の「商品から選ぶ / 自由入力」パターンを他の類似機能にも適用しているか？

## 一貫性
- 類似機能（物販↔回数券、予約↔カルテ等）が同じUIパターンを使っているか？
- 同じアクション（作成・編集・削除）のボタンデザインが統一されているか？
- モード切替UIのデザインが全箇所で同一か？

## 空状態・エラー状態
- データがゼロの時に案内テキスト + 作成ボタンがあるか？
- エラー時に `ErrorAlert` で具体的な原因を表示しているか？
- 成功時に `FlashToast` / `Toast` でフィードバックがあるか？

## 動線の重複と役割分担
- 同じ機能に複数のエントリーポイントがある場合、それぞれの役割が明確か？
- ヘルプテキストで「別の方法」を案内しているか？
- 例: 回数券は「カルテ作成時のインライン販売」と「顧客詳細からのスタンドアロン登録」の2動線
  - インライン: 来店中に販売する場面
  - スタンドアロン: 事前登録・過去分の追加
