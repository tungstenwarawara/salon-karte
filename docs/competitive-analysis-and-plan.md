### 各サービス詳細

#### Bionly（ビオンリー）
- iPad専用のオールインワン型
- FREEプラン: スタッフ1名・レシート1,000件まで
- 有料: 年契約月払い9,800円（税抜）、月契約29,000円
- 電子カルテ（手書き対応）、POS、予約、売上分析
- 顧客アプリ「CHEERBE」無料付帯

#### LiME（ライム）
- 売上連動型の変動料金（美容師向けスマホアプリ）
- 無料: カルテ5枚/人、予約5件/月
- カルテ管理プラン: 380~1,980円（売上連動）
- ネット予約プラン: 980~5,780円（売上連動）
- エステ特化テンプレートはなし

#### KaruteKun（カルテくん）
- カルテ特化型スマホアプリ
- シングル: 5,500円/月（LINE連携なし）
- シングル+LINE: 11,000円/月
- 30日無料トライアルあり

#### Square 予約
- 汎用型予約+POSレジ
- フリー: 0円、プラス: 3,300円/月
- カルテ機能なし（予約・決済特化）

#### SALON BOARD
- HPB掲載が前提（実質月2.5万円~）
- SALON BOARD自体は無料
- 簡易カルテのみ（写真管理なし）

#### リザービア
- 予約一元管理特化（高価格帯）
- Basic S: 21,000円/月 + 初期100,000円
- LINE連携: +4,500円/月

#### サロンアンサー
- 本格POS+カルテ（中~大規模向け）
- 月額9,800円~ + 初期130,000円
- LINE: +6,800円/月

#### coming-soon
- 集客自動化に強い
- 月額14,300円~ + 初期33,000円
- 2つの特許技術（来店行動学習・自動スタッフ振り分け）

## 3. 料金プラン設計

### 基本プラン: 2,980円/月（税込）

| 含まれる機能 | 競合との優位性 |
|------------|-------------|
| 電子カルテ（枚数無制限） | LiME無料は5枚/人制限 |
| ビフォーアフター写真（5GB） | エステ特化の差別化ポイント |
| 顧客管理（人数無制限） | Bionly FREEはスタッフ1名制限 |
| 予約管理（営業日設定・空き時間表示） | KaruteKunと同等 |
| 施術メニュー管理 | 複数メニュー選択対応 |
| 物販購入履歴 | **競合の大半が未対応** |
| 回数券・コース管理 | **エステの売上の柱。ほぼ全競合が未対応** |
| 月間売上集計（施術+物販+回数券） | 3軸の集計は独自 |
| 離脱顧客アラート | 60日/90日で自動検出 |
| 来店分析（基本） | 来店回数・間隔・最終来店日 |

**価格妥当性**: KaruteKun（5,500円）の半額以下で同等+α（回数券・物販）

### お試しプラン: 0円
- 顧客10件まで
- 写真保存・予約管理なし

### オプション機能

## 4. salon-karteの3つの差別化ポイント

### 1. エステ特化のカルテ機能
- 競合の大半は美容室（ヘアサロン）向け
- ビフォーアフター写真管理、施術前の状態記録、肌の経過記録
- エステサロンに必要な情報項目を標準装備

### 2. 初期費用0円 + 月額2,980円
- リザービア: 初期10万 + 月2.1万
- サロンアンサー: 初期13万 + 月1万
- coming-soon: 初期3.3万 + 月1.4万
- → salon-karteは0円スタートで圧倒的に導入しやすい

### 3. 回数券・物販管理が基本機能
- エステでは回数券が売上の柱
- 物販（化粧品・サプリ等）も重要な収益源
- これらが基本料金に含まれるのはほぼ唯一
